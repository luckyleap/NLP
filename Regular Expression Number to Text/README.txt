HOW TO RUN THE PROGRAM:
	In the terminal, enter: 'python assig1.py [input.txt] [output.txt]'

	- [input.txt] is the input file containing all the phrases that will be translated
	- [output.txt] is the output file that will contain all the translated phrases

BUGS:
	- Does not capitalize the translated number, example: 'one apple fell from the tree'

AUGMENTATIONS:
	- Handles all the basic requirements
	- Handles decimal in dollars
	- Handles different dollar formats
	- Handles decimal in percent
	- Handles ordinal
	- Handles different date formats
	- Handles commas in numbers
